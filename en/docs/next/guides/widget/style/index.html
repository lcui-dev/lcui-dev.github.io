Таит╝Ј

Т»ЈСИфу╗ёС╗ХжЃйТюЅУЄфт«џС╣ЅТаит╝ЈсђЂти▓тї╣жЁЇТаит╝ЈсђЂТюђу╗ѕТаит╝Јтњїти▓У«Ау«ЌТаит╝ЈтЁ▒тЏЏт╝аТаит╝ЈУАе№╝їтЁХСИГти▓У«Ау«ЌТаит╝Јућ▒Тюђу╗ѕТаит╝ЈУ«Ау«ЌУђїТЮЦ№╝їУђїТюђу╗ѕТаит╝ЈтѕЎТў»ти▓тї╣жЁЇТаит╝ЈСИјУЄфт«џС╣ЅТаит╝Јтљѕт╣ХУђїТЮЦуџёсђѓ

тйЊу╗ёС╗ХуџёУЄфт«џС╣ЅТаит╝ЈсђЂуіХТђЂсђЂу╗ЊТъёС╗ЦтЈіТаит╝Јт║ЊТюЅТћ╣тіеуџёТЌХтђЎС╝џУДдтЈЉТаит╝ЈУ«Ау«Ќ№╝їУ«Ау«ЌУ┐ЄуеІТў»тЁѕС╗јТаит╝Јт║ЊСИГТЪЦУ»бСИју╗ёС╗Хтї╣жЁЇуџёТаит╝ЈУАе№╝їуёХтљјтљѕт╣ХСИ║СИђт╝аТаит╝ЈУАеСйюСИ║у╗ёС╗Хуџёти▓тї╣жЁЇТаит╝Ј№╝їС╣Ітљјт░єти▓тї╣жЁЇТаит╝ЈСИјУЄфт«џС╣ЅТаит╝Јтљѕт╣ХТѕљСИђт╝аТаит╝ЈУАеСйюСИ║Тюђу╗ѕТаит╝Ј№╝їТюђтљјт░єТюђу╗ѕТаит╝ЈУйгТЇбСИ║ти▓У«Ау«ЌТаит╝Ј№╝їС╗ЦСЙЏтИЃт▒ђт╝ЋТЊјтњїТИ▓ТЪЊт╝ЋТЊјСй┐ућесђѓ

жЎцУЄфт«џС╣ЅТаит╝Јтцќ№╝їтЁХт«ЃТаит╝ЈУАежЃйТў»тЈфУ»╗уџё№╝їСйаСИЇт║ћУ»ЦС┐«Тћ╣т«ЃС╗гсђѓ

УЄфт«џС╣ЅТаит╝ЈРђІ

{
    float opacity = w->computed_style.opacity;
    
    if (w->computed_style.visible) {
        if (opacity > 0) {
            opacity -= 0.01;
        }
        if (opacity > 0) {
            Widget_SetOpacity(w, opacity);
        } else {
            Widget_Hide(w);
        }
    }
}
